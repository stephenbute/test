How to view locally
-------------------

Open `index.html` in your browser, or serve the folder with a simple HTTP server:

```bash
cd /path/to/repo
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```

Notes
-----

- Edit any `.html` file in the repo and refresh the browser to see changes.
- The files are static HTML pages intended as a small demo/site skeleton.
